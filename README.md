# ATM Simulator (Python)

## Description
This project simulates basic ATM operations using Python.
It allows the user to enter a PIN, check account balance,
and withdraw money.

## Features
- PIN verification
- Balance enquiry
- Cash withdrawal
- Error handling for wrong PIN and insufficient balance

## Technologies Used
- Python 3

## How to Run
1. Download `atm_simulator.py`
2. Open terminal / command prompt
3. Run:
   python atm_simulator.py

## Learning Outcome
- Conditional statements
- User input handling
- Basic control flow logic
